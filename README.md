## Задание 7: Творческий проект — Программа «Список учеников»

### Что делает программа

Это программа для учителя, чтобы вести список учеников в классе. Можно добавлять учеников (имя + оценка), сортировать их по оценкам (от лучших к худшим или наоборот), и искать конкретного ученика по имени.

### Какие алгоритмы используются

1. **Линейный поиск** — когда ищу ученика, программа проходит по всему списку и сравнивает каждое имя с тем, что ввел учитель. Это просто и понятно, работает даже если список не отсортирован.

2. **Сортировка пузырьком** — когда нужно отсортировать учеников по оценкам, я использую пузырёк. Программа идёт по списку, сравнивает соседние оценки и меняет их местами, если нужно. На маленьком списке (30 учеников) это нормально работает.

### Примерный интерфейс

```
===== СПИСОК УЧЕНИКОВ =====
-----------------------------

1. Добавить ученика
2. Показать всех
3. Найти ученика
4. Сортировать по оценке
5. Выход

Выберите действие: 3

Введите имя ученика: Анна

Поиск... Анна найдена!
Имя: Анна
Оценка: 5

Всего проверено: 5 учеников из 28
```

```
===== СПИСОК УЧЕНИКОВ =====

1. Добавить ученика
2. Показать всех
3. Найти ученика
4. Сортировать по оценке
5. Выход

Выберите действие: 4

Сортировка по убыванию оценок:
1. Анна - 5
2. Борис - 4
3. Вася - 4
4. Галя - 3
...

Сделано 378 сравнений
```

### Почему выбрал именно эти алгоритмы

**Линейный поиск:**
- Простой в реализации, я его понял сразу
- Список учеников небольшой (30 человек), разница в скорости не критична
- Не нужно сортировать список каждый раз для поиска

**Сортировка пузырьком:**
- Я её уже знаю и могу написать сам без ошибок
- Для 30 учеников это достаточно быстро (меньше секунды)
- Легко объяснить как она работает преподавателю
- Можно визуально показать как элементы меняются местами
